# I'm Reading Here!

I'm Reading Here! is an open source project that enables its users to search for, save to read, and track the books that they have read.
This is the back end service to facilitate the mobile and web apps needs.

## Prerequisites

To have this work locally you will need to create a `.env` file and enter a few keys to be used, including an API key for Google.

```.env
GOOGLE_API_KEY=[YOUR_API_KEY]
```
